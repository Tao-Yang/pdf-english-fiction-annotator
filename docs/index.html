PDF 英文小说中文注释工具

打开网页就能用。自动为英文原版小说的生词、短语和习语，在页面右侧空白处加上简洁的中文释义，不遮挡原文。

🚀 在线体验（Web App） ▶ 如何上线（一次性）

若“在线体验”打不开，说明服务还未部署——按下方“如何上线”完成一次性设置即可。

三步开始使用

  1. 打开 在线 Web App（若尚未上线，先按下方“如何上线”部署），无需安装任何软件。
  2. 上传一个英文 PDF，挑选难度级别（CEFR A1–C2），如需可填写正文起始页。
  3. 点击“开始注释”，稍候即可下载带中文注释的新 PDF，原书排版与目录链接保持不变。

如何上线（一次性设置）

Web App 需要运行在你自己的（免费）服务上。最简单的方式是 Hugging Face Spaces，仓库已配好自动部署流程，你只需做一件事：添加一个令牌。

  1. 创建令牌：打开 huggingface.co/settings/tokens → New token → 类型选 Write → 复制。
  2. 存入 GitHub：仓库 Settings → Secrets and variables → Actions → New repository secret，名称填 HF_TOKEN，值粘贴令牌。
  3. 触发部署：仓库 Actions 页 → “Deploy to Hugging Face Space” → Run workflow。稍候即会自动创建 Space 并构建上线。

完成后，上方“在线体验”按钮即可直接打开；之后每次推送代码都会自动重新部署。

特点

不遮挡原文

向页面右侧扩展出空白页边，中文释义整齐排列，绿色下划线标出对应词语。

按难度筛词

基于词频与 CEFR 级别，只注释“值得注释”的生词、短语和习语，避免满页翻译。

任意阅读器可见

中文以图片形式渲染嵌入，不依赖阅读器的中文字体，手机、平板、电脑都能正常显示。

保留导航

目录内部链接与书签（大纲）会被重新写入，点击目录依然可以跳转。

喜欢命令行 / 想自建服务？

本地跑网页版（Gradio）：

pip install -r webapp/requirements.txt
python webapp/app.py        # 打开 http://localhost:7860

或用 Docker 一键部署到 Hugging Face Spaces / Render / Railway：

docker build -t pdf-annotator .
docker run -p 7860:7860 pdf-annotator

纯命令行批量处理：

pip install -e .
python -m scripts.download_ecdict          # 首次下载词典
annotate-fiction novel.pdf --level B2

更多参数见 README。
